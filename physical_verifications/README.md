# Physical Verifications

Companion documents to the main paper. Each document records a concrete comparison between the DS framework's predictions and physical data, with full numerical results, honest assessment, and identified next steps.

**Standard:** Zero free parameters. Every prediction is a consequence of H=3, K*=7/30, Born floor 1/27, and the gauge group's Cartan matrix. Nothing is fitted. Deviations are reported in sigma. Failures are documented as clearly as successes.
